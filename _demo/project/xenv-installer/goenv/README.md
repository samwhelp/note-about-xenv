### prepare

``` sh
$ ./prepare.sh
```

or

``` sh
$ make prepare
```


### install

``` sh

```

or

``` sh
$ make install
```

### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

``` sh
$ make config-install
```
